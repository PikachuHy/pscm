# Krabby

一个简单的打字软件.开发代号“大钳蟹”
![Krabby](http://s1.52poke.wiki/wiki/thumb/a/a7/098Krabby.png/300px-098Krabby.png)
## 大钳蟹简介
大钳蟹（日文︰クラブ，英文︰Krabby）是水属性宝可梦。

外貌
大钳蟹有点像招潮蟹，大钳蟹的体色由红色和白色组成，头部的上半部分和两只蟹爪为红色，其余部分为白色。头顶有两个突起，下颌部分有着状似牙齿的凸起。六条肢体（包含钳子）都分节，它似乎还可以吃。

性别差异
大钳蟹没有性别差异。

特殊能力
性情
落日时会聚集在一起吐泡沫。

栖息地
沙滩的洞中。
